# furniCraft

Welcome to **furniCraft** – A modern furniture design and crafting showcase.

## 🎨 About

furniCraft is a web application dedicated to showcasing beautiful furniture designs and crafting techniques. Our platform brings together creativity and functionality to present high-quality furniture pieces with detailed descriptions and visual galleries.

## 🌟 Features

- **Responsive Design** – Mobile-friendly interface that works on all devices
- **Beautiful UI** – Modern styling with CSS and SCSS
- **Interactive Elements** – Smooth user experience with JavaScript interactivity
- **Portfolio Showcase** – Display furniture collections and designs
- **Easy Navigation** – Intuitive layout for browsing furniture pieces

## 🛠️ Tech Stack

- **HTML** – Semantic markup and structure
- **CSS & SCSS** – Advanced styling and responsive design
- **JavaScript** – Dynamic functionality and interactivity

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Explore the furniture showcase

## 💡 Usage

Simply navigate through the website to browse our collection of carefully curated furniture designs. Each piece includes detailed information and high-quality images.
